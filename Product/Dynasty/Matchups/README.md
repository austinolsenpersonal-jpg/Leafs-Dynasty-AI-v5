# Toronto Maple Leafs – Matchup Gameplans

This directory stores all opponent-specific matchup gameplans created for the 2024–25 season and beyond.

Each gameplan follows the official Leafs Dynasty AI format:
- Full NHL 25 systems (in exact menu order)
- Line strategies with 0–10 sliders
- Defensive pair strategies
- Period-by-period plan
- Opponent line-match targeting
- Defense pair vulnerabilities
- Special teams plan
- Goalie gameplan
- Bench triggers (fast switches)
- Optional printable/PDF version

## File Naming Convention
Use the following structure for all gameplan files:

YYYY-MM-DD_vs_<TEAM>_Gameplan.md

Examples:
2024-12-27_vs_DET_Gameplan.md
2024-12-28_vs_WSH_Gameplan.md
2025-01-04_vs_BOS_Gameplan.md
